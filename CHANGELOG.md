# Changelog

All notable changes to this project are documented in this file.

## [1.0.0] - 2026-07-04

The original prototype concept has grown into a complete collection facade library with optimized fuzzy indexes, typed public APIs, documentation, test coverage, and benchmark-backed strategy decisions.

## [0.1.0] - 2024-02-29

Initial prototype release validating the core idea: Python collection-like objects with built-in fuzzy lookup powered by RapidFuzz.
