# Third-party example datasets

This directory contains third-party datasets used only for examples, documentation, and local experimentation with `rapidfuzz-collections`.

These datasets are not part of the Python package distribution and are not covered by this project's source code license.

## Included datasets

* Structured Fodors-Zagats
* Structured Amazon-Google Products
* Dirty DBLP-ACM

## Source

The datasets were obtained from the Zenodo record:

George Papadakis, "Datasets for Supervised Matching in Clean-Clean Entity Resolution", Zenodo, DOI: [10.5281/zenodo.8164151](https://doi.org/10.5281/zenodo.8164151).

The datasets are also commonly associated with the DeepMatcher / Magellan entity matching benchmark collection.
